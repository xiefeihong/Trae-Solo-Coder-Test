# 🔥 森林冰火人 ❄️ - Fire & Ice

基于 Vue 3 + TypeScript + Canvas 开发的经典解谜游戏《森林冰火人》网页版。

## 🎮 游戏特性

### 核心玩法
- **双人协作**：控制火娃（红色）和冰娃（蓝色）两个角色
- **元素交互**：
  - 🔥 火娃：可以安全通过水坑，但碰到岩浆会死亡；只能收集火宝石
  - ❄️ 冰娃：可以安全通过岩浆，但碰到水坑会死亡；只能收集冰宝石
- **目标**：收集所有宝石后，两个角色都到达各自的传送门即可过关

### 控制模式
1. **双人模式**（默认）：
   - 火娃：↑ ↓ ← → 方向键移动
   - 冰娃：W A S D 键移动

2. **单人模式**：
   - 使用方向键或 WASD 控制当前角色
   - 按 Tab 键切换控制角色
   - 按 R 键重置当前关卡

### 地图元素
| 元素 | 说明 |
|------|------|
| 🧱 墙壁 | 不可通过 |
| 🌋 岩浆 | 火娃危险，冰娃安全 |
| 💧 水坑 | 冰娃危险，火娃安全 |
| 💎 火宝石 | 火娃可收集 |
| 💎 冰宝石 | 冰娃可收集 |
| 🌀 火传送门 | 火娃终点 |
| 🌀 冰传送门 | 冰娃终点 |
| ⬜ 压力板 | 踩下后可打开机关门 |
| 🚪 机关门 | 需压力板激活才能通过 |

### 关卡设计
游戏包含 5 个精心设计的关卡：
1. 新手教程 - 基础操作演示
2. 水火不容 - 元素地形挑战
3. 机关重重 - 压力板与门的配合
4. 协同合作 - 复杂机关解谜
5. 终极挑战 - 综合难度考验

## 🛠️ 技术栈

- **框架**：Vue 3 (Composition API + `<script setup>`)
- **语言**：TypeScript
- **渲染**：HTML5 Canvas 2D
- **构建工具**：Vite
- **状态管理**：Vue 3 响应式系统 (ref/reactive)

## 🎯 游戏玩法说明

1. **开始游戏**：打开网页后自动进入第一关
2. **移动角色**：使用对应按键移动角色
3. **收集宝石**：移动到对应颜色的宝石位置自动收集
4. **触发机关**：踩在压力板上可以打开机关门
5. **到达终点**：收集完所有宝石后，两个角色都站在各自的传送门上即可过关
6. **切换关卡**：过关后自动进入下一关，也可使用界面按钮手动切换

## 💡 游戏提示

- 角色无法穿过彼此
- 注意观察地形，选择正确的路线
- 压力板需要一直踩着才能保持门打开
- 善用"重置关卡"功能（R键）重新开始
- 在双人模式下，建议与朋友一起协作游戏

## 🔧 扩展开发

### 添加新关卡

在 `src/levels.ts` 中添加新的关卡数据：

```typescript
{
  name: '关卡名称',
  map: [
    // 使用 TileType 枚举定义二维数组
    [TileType.WALL, TileType.GROUND, ...],
    // ...
  ],
  fireStart: { x: 1, y: 1 },    // 火娃起始位置
  iceStart: { x: 10, y: 1 },     // 冰娃起始位置
  totalFireGems: 2,              // 火宝石总数
  totalIceGems: 2,               // 冰宝石总数
}
```

### 修改游戏参数

在 `src/composables/useFireIce.ts` 中可调整：
- `TILE_SIZE`：格子大小（默认 48px）
- 角色移动逻辑
- 胜利/失败条件判定

## 📝 快捷键一览

| 快捷键 | 功能 |
|--------|------|
| R | 重置当前关卡 |
| Tab | 单人模式下切换控制角色 |

| W A S D | 双人模式控制冰娃 / 单人模式控制当前角色 |
