# 俄罗斯方块 (Vue 3 + TypeScript)

一个功能完整的俄罗斯方块游戏，使用 Vue 3 Composition API 和 TypeScript 开发。

## 功能特性

### 核心游戏玩法
- 标准 10x20 游戏网格
- 7 种不同形状的方块（I、O、T、L、J、S、Z），每种有独特颜色
- 方块旋转和碰撞检测
- 幽灵方块（显示方块落底位置）
- 连续消除多行检测

### 计分系统
- 单行消除：100 分 × 等级
- 双行消除：300 分 × 等级
- 三行消除：500 分 × 等级
- 四行消除（Tetris）：800 分 × 等级
- 硬落加分：每格 2 分

### 等级系统
- 每消除 10 行提升一级
- 等级越高，下落速度越快
- 初始间隔 500ms，每级减少 50ms，最低 100ms

### 高级功能
- **暂存区（Hold）**：按 C 键暂存当前方块，每回合只能使用一次
- **多个下一方块预览**：显示接下来 3 个方块
- **最高分记录**：使用 localStorage 保存最高分
- **游戏暂停**：按 P 或 Esc 暂停/继续游戏

## 操作说明

### 键盘操作
- `←` `→`：左右移动方块
- `↑`：旋转方块
- `↓`：加速下落（按住持续加速）
- `空格`：硬落（直接落到底部）
- `C`：暂存方块
- `P` / `Esc`：暂停/继续游戏

### 移动端触控
- 屏幕下方提供虚拟方向键
- 支持旋转、移动、加速、硬落和暂存功能
- 响应式设计，自动适配不同屏幕尺寸

## 技术栈
- **Vue 3**：Composition API + `<script setup>`
- **TypeScript**：类型安全
- **Vite**：快速构建工具
- **Canvas 2D**：游戏渲染

- 响应式布局，支持桌面端和移动端

## 开发说明
游戏使用 `requestAnimationFrame` 实现游戏循环，确保平滑的动画效果。所有游戏状态都通过 Vue 的响应式系统管理，UI 会自动根据状态变化更新。

碰撞检测使用墙踢（Wall Kick）算法，让方块在边界处也能正常旋转。方块锁定后会检查完整行并消除，同时上方方块会自动下落。
