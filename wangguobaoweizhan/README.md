# 王国保卫战 - 塔防游戏

一个基于 Vue 3 + TypeScript + Canvas 2D 的《王国保卫战》风格塔防游戏。

## 功能特性

### 核心玩法
- ✅ 敌人沿固定路线从起点走向终点
- ✅ 玩家在路线旁建造防御塔阻止敌人
- ✅ 敌人到达终点扣除生命值，生命值归零游戏失败
- ✅ 击败敌人获得金币，用于建造和升级防御塔

### 防御塔类型
- **箭塔**: 攻速快、伤害中等、射程较远、可攻击飞行单位
- **兵营**: 召唤士兵阻挡敌人，士兵有生命值并可升级
- **法师塔**: 攻速慢、伤害高、可破甲、可攻击飞行单位
- **炮塔**: 范围伤害、攻速慢、适合对付成群敌人

### 敌人类型
- **哥布林**: 血量低、移速快
- **兽人**: 血量中等、移速慢
- **巨魔**: 血量高、移速极慢、高价值
- **飞行敌人**: 无视地面阻挡，只能用箭塔或法师塔攻击

### 升级系统
- 每座塔可升级 2 次（共 3 级）
- 升级提升伤害、攻速、射程等属性
- 塔可以出售获得部分金币

### 波次系统
- 10 波敌人，难度递增
- 每波结束后获得奖励金币
- 波次间可准备建造和升级防御塔

### 额外功能
- 游戏速度调节（1x / 2x / 3x）
- 暂停功能
- 敌人血条显示
- 塔攻击范围显示

## 技术栈

- **框架**: Vue 3 (Composition API + `<script setup>`)
- **语言**: TypeScript
- **渲染**: Canvas 2D API
- **状态管理**: Pinia
- **构建工具**: Vite
- **游戏循环**: requestAnimationFrame

## 游戏操作说明

1. **开始游戏**: 点击顶部的"🎮 开始游戏"按钮
2. **建造防御塔**:
   - 点击右侧面板中的塔类型
   - 鼠标移动到地图上的高亮可建造位置
   - 点击鼠标建造防御塔
3. **升级/出售塔**:
   - 点击已建造的防御塔
   - 在弹出的面板中选择升级或出售
4. **控制游戏速度**: 点击顶部的速度按钮切换 1x/2x/3x
5. **暂停游戏**: 点击暂停按钮暂停游戏

## 游戏策略提示

- 箭塔适合对付快速敌人和飞行敌人
- 兵营适合放在关键路口阻挡敌人
- 法师塔对付高护甲敌人效果最好
- 炮塔在敌人密集的位置效果最佳
- 合理利用升级可以大幅提升防御效率
- 注意金币的平衡使用，不要过早升级

## 开发说明

### 核心游戏循环
游戏使用 `requestAnimationFrame` 实现主循环，每帧执行以下操作：
1. 更新敌人位置和状态
2. 塔的目标选择和攻击判定
3. 投射物的飞行和碰撞检测
4. 士兵的 AI 和战斗
5. 渲染所有游戏元素

### 地图系统
- 网格化地图，每个格子有地形类型（草地、道路、可建造等）
- 敌人沿预定义路径点插值移动
- 使用 AABB 碰撞检测处理建造位置

### 状态管理
使用 Pinia 管理全局游戏状态（金币、生命值、波次、游戏状态等），游戏引擎的状态与 Pinia store 同步。
