# Vue 3 塔防游戏开发提示词

## 项目概述
开发一个基于 Vue 3 + TypeScript + Canvas 2D 的《王国保卫战》风格塔防网页游戏。

---

## 一、核心玩法

### 1.1 基础规则
- 敌人沿固定路线从起点走向终点
- 玩家在路线旁建造防御塔阻止敌人
- 敌人到达终点扣除生命值，生命值归零游戏失败
- 击败敌人获得金币，用于建造和升级防御塔

### 1.2 防御塔类型（至少4种）
| 塔类型 | 特点 |
|--------|------|
| **箭塔** | 攻速快、伤害中等、射程较远、可攻击飞行敌人 |
| **兵营** | 召唤士兵阻挡敌人，士兵有生命值并可升级 |
| **法师塔** | 攻速慢、伤害高、可破甲、可攻击飞行敌人 |
| **炮塔** | 范围伤害、攻速慢、适合对付成群敌人 |

### 1.3 敌人类型（至少4种）
| 敌人类型 | 特点 |
|----------|------|
| **哥布林** | 血量低、移速快 |
| **兽人** | 血量中等、移速慢 |
| **巨魔** | 血量高、移速极慢、高价值 |
| **飞行敌人** | 无视地面阻挡，只能用箭塔或法师塔攻击 |

### 1.4 升级系统
- 每座塔可升级2次（共3级）
- 升级提升伤害、攻速、射程等属性

### 1.5 英雄系统（简化版）
- 一个可移动的英雄单位
- 可手动控制或自动攻击附近敌人

### 1.6 波次系统
- 敌人按波次出现
- 每波结束后可进入下一波
- 波次间有准备时间

## 三、地图设计

### 3.1 网格化地图
- 尺寸：20x15 或 15x15 格子
- 预置1-2张地图

### 3.2 地图元素
| 元素 | 说明 |
|------|------|
| 路径点 | 敌人移动路线 |
| 可建造塔位 | 高亮显示 |
| 起点 | 敌人出生点 |
| 终点 | 敌人目标点 |
| 草地 | 可造塔 |
| 道路 | 不可造塔 |
| 河流 | 飞行敌人可通过 |

---

## 四、战斗逻辑

### 4.1 塔攻击逻辑
- 每帧检测射程内敌人
- 冷却结束后选择目标并攻击
- 目标选择策略（可配置）：优先最近/最远/血量最低

### 4.2 兵营逻辑
- 在塔周围一定范围内生成士兵
- 士兵自动寻找附近敌人进行近战攻击
- 死亡后在塔位置重生（有冷却）

### 4.3 投射物系统
- 攻击时生成子弹/箭矢
- 飞行到敌人位置造成伤害
- 炮塔有爆炸范围伤害

### 4.4 目标选择策略
- 箭塔：优先打低血量敌人
- 炮塔：优先打密集区域

## 五、UI界面

### 5.1 顶部状态栏
- 金币数量
- 生命值
- 当前波次
- 下一波倒计时/按钮

### 5.2 建造面板
- 位置：右侧或底部
- 塔图标列表
- 点击塔图标进入建造模式

### 5.3 塔操作面板
- 点击已建造的塔显示
- 升级按钮（显示升级费用）
- 出售按钮（显示回收价格）

### 5.4 游戏结束面板
- 显示最终波次
- 重新开始按钮

---

## 六、交互流程

1. **建造流程**：点击建造面板中的塔图标 → 进入建造模式（鼠标变十字）→ 点击可建造格子 → 消耗金币建造
2. **塔操作**：点击已建造的塔 → 弹出升级/出售菜单
3. **波次控制**：点击"开始下一波"按钮 → 生成敌人
