# 🏗️ 建筑队大战僵尸 🧟

一个基于 Vue 3 + TypeScript + Canvas 开发的塔防类网页游戏，类似植物大战僵尸玩法。

## 🎮 游戏特性

### 核心玩法
- **左侧建造建筑**：玩家在左侧区域建造各种建筑
- **自动生产士兵**：建筑自动生产对应类型的士兵
- **右侧僵尸进攻**：僵尸从右侧不断向左移动
- **自动战斗**：士兵自动攻击遇到的僵尸
- **保护基地**：阻止僵尸到达左侧基地

### 建筑类型（5种）
1. **水泥厂** - 生产基础工人，攻击力低、造价便宜、生产速度快
2. **木工坊** - 生产木匠，中等攻击力，可对空攻击（对付飞行僵尸）
3. **钢铁厂** - 生产铁匠，高攻击力、高生命值、生产速度慢、造价高
4. **爆破实验室** - 生产投弹兵，范围伤害，适合对付成群僵尸
5. **维修站** - 生产维修工，不能攻击，但可以修复受损建筑

### 僵尸类型（5种）
1. **普通僵尸** - 血量和速度中等，最基础的敌人
2. **路障僵尸** - 高生命值，移动慢，需要持续输出才能击杀
3. **飞行僵尸** - 无视地面单位，只能被对空单位攻击（木匠），优先攻击建筑
4. **自爆僵尸** - 生命值低但移动速度快，靠近建筑或士兵会自爆造成范围伤害
5. **BOSS僵尸** - 每5波出现一次，极高生命值和攻击力

### 游戏系统
- **资源系统**：初始500资源，击败僵尸获得奖励，每波结束获得波次奖励
- **升级系统**：每个建筑可升级3次，提升生产速度和士兵属性
- **波次系统**：共10波，难度递增，波次之间有准备时间
- **暂停功能**：随时暂停/继续游戏
- **出售建筑**：出售已建造的建筑获得50%资源返还

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000 开始游戏

### 构建生产版本
```bash
npm run build
```

## 🎯 操作指南

1. **建造建筑**：点击左侧建筑面板中的建筑，然后在战场上点击放置（80-400像素范围内）
2. **升级建筑**：点击已建造的建筑，然后点击"升级"按钮
3. **出售建筑**：点击已建造的建筑，然后点击"出售"按钮
4. **暂停游戏**：点击右上角的"暂停"按钮
5. **提前开始波次**：点击波次信息栏的"立即开始"按钮
6. **重新开始**：点击右上角的"重开"按钮

## 💡 游戏策略

1. **前期**：多建造水泥厂快速生产工人，形成数量优势
2. **对空防御**：务必建造木工坊来对付飞行僵尸
3. **中期**：建造钢铁厂和爆破实验室增强火力
4. **治疗**：建造维修站修复受损建筑
5. **升级**：优先升级高等级建筑，提升生产效率

## 🛠️ 技术栈

- **框架**：Vue 3 (Composition API)
- **语言**：TypeScript
- **渲染**：Canvas 2D
- **构建工具**：Vite
- **状态管理**：Vue 3 响应式系统

```
│   └── renderer.ts   # Canvas 渲染
├── components/       # Vue 组件
│   └── GameCanvas.vue
├── types.ts          # 类型定义
├── App.vue           # 主应用组件
├── main.ts           # 入口文件
└── style.css         # 全局样式
```
